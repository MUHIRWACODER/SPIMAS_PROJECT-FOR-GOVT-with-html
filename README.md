# SPIMAS - Smart Public Infrastructure Monitoring & Accountability System

## 🏛️ Project Overview

SPIMAS is a government transparency and accountability platform designed to empower Rwandan citizens with visibility into public infrastructure projects and fund allocation. The system enables real-time monitoring of roads, bridges, schools, hospitals, and other public infrastructure while promoting accountability among government officials and contractors.

**Developed by:** Jules Co. in partnership with the Government of Rwanda

---

## 🎯 Mission & Goals

Our mission is to ensure every Rwandan citizen can access transparent information about where public funds are allocated and how they are being used.

### Core Goals:
- **🔍 Transparency:** Making government budgets and project allocations visible to the public
- **⚖️ Accountability:** Holding officials and contractors responsible for project delivery
- **📊 Monitoring:** Real-time tracking and progress updates on infrastructure projects

## 🛠️ Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Edge, Safari)
- Internet connection
- No backend server required (static HTML/CSS/JavaScript)

### Installation & Usage

1. **Clone the Repository**
   ```bash
   git clone https://github.com/MUHIRWACODER/SPIMAS_PROJECT-FOR-GOVT-with-html.git
   cd SPIMAS_PROJECT-FOR-GOVT-with-html
   ```

2. **Open in Browser**
   - Simply open `index.html` in your web browser
   - Or deploy to a web server (GitHub Pages, Netlify, etc.)

3. **Navigate the System**
   - Start at the Home Page (index.html)
   - Select your role: Citizen, Engineer, or Contractor
   - Login with your credentials
   - Access role-specific features

### To Deploy on GitHub Pages
1. Go to repository Settings
2. Enable GitHub Pages from the `main` branch
3. Your site will be available at: `https://MUHIRWACODER.github.io/SPIMAS_PROJECT-FOR-GOVT-with-html/`
